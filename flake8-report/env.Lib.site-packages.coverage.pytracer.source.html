50 # Because of poor implementations of trace-function-manipulating tools,
51 # the Python trace function must be kept very simple. In particular, there
52 # must be only one function ever set as the trace function, both through
53 # sys.settrace, and as the return value from the trace function. Put
54 # another way, the trace function must always return itself. It cannot
55 # swap in other functions, or return None to avoid tracing a particular
56 # frame.
57 #
58 # The trace manipulator that introduced this restriction is DecoratorTools,
59 # which sets a trace function, and then later restores the pre-existing one
60 # by calling sys.settrace with a function it found in the current frame.
61 #
62 # Systems that use DecoratorTools (or similar trace manipulations) must use
63 # PyTracer to get accurate results. The command-line --timid argument is
64 # used to force the use of this tracer.
