44 """Collects trace data.
45  
46 Creates a Tracer object for each thread, since they track stack
47 information. Each Tracer points to the same shared data, contributing
48 traced data points.
49  
50 When the Collector is started, it creates a Tracer for the current thread,
51 and installs a function to create Tracers for each new thread started.

227 # should_trace_cache is a strictly growing key: once a key is in
